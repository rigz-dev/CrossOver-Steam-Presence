CrossOver Steam Presence

Detects whether CrossOver is running on Mac, fetches the Steam user's current game, updates Discord activity.

Requires Steam API key and SteamID64

Get Steam API Key Request: https://steamcommunity.com/dev/apikey

Get Steam UserID
  1. Open the Steam app and log in.
  2. Click on your profile name in the top right corner.
  3. Select Account details.
  4. Your 17-digit SteamID64 is located under your account name.
